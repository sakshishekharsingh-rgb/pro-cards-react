# 💼 Freelancer Profile Cards

A responsive profile card UI built using **React.js** and **CSS**. This project displays freelancer and company profiles with different icons, skills, pricing, and availability status.

## 🚀 Features

- Responsive Profile Cards
- React Functional Components
- Props for Dynamic Data
- Conditional Rendering
- Lucide React Icons
- Flexbox Layout
- Modern Card Design
- Reusable Components

## 🛠️ Technologies Used

- React.js
- JavaScript (ES6)
- CSS3
- HTML5
- Lucide React

## 📂 Project Structure

```
src/

│   ├── Card.jsx

```
